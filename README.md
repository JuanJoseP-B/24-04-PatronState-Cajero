# StateCash: ATM Simulator (State Pattern)

Este es un simulador de cajero automático profesional desarrollado para demostrar la implementación pura del **Patrón de Diseño State**.

## 🚀 Cómo Funciona
1. **Backend (Java/Spring Boot):** Gestiona la lógica de estados (Idle, CardInserted, PinValidation, etc.) y es la fuente de verdad.
2. **Frontend (React/Vite):** Interfaz reactiva que se sincroniza en tiempo real con el backend.

## 🛠️ Ejecución rápida
1. Iniciar el backend desde la clase `AtmApplication.java`.
2. Ejecutar `npm install` y `npm run dev` en la carpeta `frontend`.
